# WinBar

**macOS の Dock を、Windows 風のタスクバーに置き換えるアプリ。**

全ディスプレイ・全仮想デスクトップに常時表示され、どのアプリが起動中・アクティブかが一目で分かります。

---

## ✨ 主な機能

- 🖥 **マルチディスプレイ対応** — すべての画面・すべての Space に表示(Dock 最大の不満を解消)
- 🔵 **状態が一目で分かる** — 起動中はピル、アクティブは長いピル + 枠線、複数ウィンドウは枚数バッジ
- 📌 **ピン留め** — 右クリックで登録、ドラッグで並び替え。区切り線をまたいでピン留め / 解除
- 🖱 **Windows と同じクリック挙動** — アクティブなアイコンをクリックでしまう、再クリックで復帰
- 🪟 **ホバープレビュー** — 全ウィンドウのサムネイル横並び。クリックでフォーカス、× で個別クローズ
- 🛡 **作業領域の確保** — 最大化やリサイズがタスクバーの下に潜り込まない(自動押し戻し)
- ⌨️ **グローバルホットキー** — ⌥Space で表示 / 非表示(変更可)
- 🕐 **システムトレイ** — 時計 + カレンダー、縦型音量スライダー(ミュート記憶付き)、常駐アプリ
- 🍅 **ポモドーロタイマー** — 直接入力・プリセット・終了時は音 + 点滅通知
- 🎨 **カラーテーマ** — 5プリセット + カスタム。サイズ・位置もスライダーで調整
- 🔍 **検索ボタン** — Spotlight をワンクリックで

## 動作環境

- macOS 13 Ventura 以降(ホバープレビューは macOS 14 以降)

## インストール

1. [Releases](../../releases) から `WinBar.dmg` をダウンロード
2. dmg を開き、**WinBar.app を Applications フォルダへドラッグ**
3. `WinBar.app` を **右クリック →「開く」** で起動
   - 初回は「開発元を確認できません」と出ます(署名なしのため)。**右クリック → 開く**で起動できます

起動すると Dock が自動的に隠れ、画面下にタスクバーが表示されます。

## 権限の許可(初回のみ)

次の**2つ**を ON にしてください。

| 権限 | 使われる機能 |
|---|---|
| **アクセシビリティ** | ウィンドウの押し戻し、ウィンドウを閉じる / 切り替え、新規ウィンドウメニュー、検索 |
| **画面収録** | ホバーで出るウィンドウプレビュー、ウィンドウタイトルの表示 |

WinBar が権限を求めると次のダイアログが出るので「システム設定を開く」を押します。

![画面収録の許可ダイアログ](https://i.gyazo.com/b801a2af0e17d91c991f6a58da0e49c2.png)

### 手順(検索窓から探すのが速いです)

1. **システム設定** を開き、検索窓に **「画面収録」** と入力
2. **「画面収録とシステムオーディオ録音」** を開き、**WinBar.app のトグルを ON**

   ![画面収録の一覧で WinBar を ON](https://i.gyazo.com/18de77977a9797948f2d32cfbba552fb.png)

3. 同じ検索結果から **「アクセシビリティ」** を開き、**WinBar.app のトグルを ON**

   ![アクセシビリティの一覧で WinBar を ON](https://i.gyazo.com/92ed5fbf690605646d720c1b55b58f7f.png)

4. 「終了して再度開く」が出たら押して再起動(出なければ手動で WinBar を終了 → 起動)

   ![終了して再度開く](https://i.gyazo.com/a1c198c931f90f1e10f5494602ca91ee.png)

> 💡 リストに WinBar.app が無いときは、下の **「+」** から `/Applications/WinBar.app` を追加できます。

## Mac 起動時に自動起動

バー左端の **⚙** → 設定 → **「ログイン時に自動起動」にチェック**。

## 基本操作

| 操作 | 動き |
|---|---|
| アイコンをクリック | アプリを前面に / アクティブなら格納(もう一度で復帰) |
| アイコンにホバー | ウィンドウのプレビュー(クリックで移動、× で閉じる) |
| アイコンを右クリック | ピン留め / 新規ウィンドウ / ウィンドウを閉じる / 終了 |
| アイコンをドラッグ | 並び替え。区切り線の左でピン留め、右で解除 |
| ⌥ + Space | タスクバーの表示 / 非表示 |
| 右端の細い帯 | デスクトップを表示 |
| トレイの ⏱ / 🔊 / 時計 | ポモドーロ / 音量 / カレンダー |

## アンインストール

バーを右クリック →「タスクバーを終了」(Dock が戻ります)→ `/Applications/WinBar.app` をゴミ箱へ。
